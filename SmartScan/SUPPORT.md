# Support

Thank you for using Smart Scanner Pro! If you need help, here are the official support channels.

## 🐛 Bug Reports
If you've found a bug or your scanner is not functioning correctly:
1. Check the [Issue Tracker](https://github.com/shovo127/Smart-Scanner-Pro/issues) to see if it has already been reported.
2. If not, open a new **Bug Report** or **Scanner Compatibility Report** using our GitHub Issue templates.
3. Always include your scanner model, OS version, and the relevant `crash_report.zip` or logs from `%APPDATA%\SmartScannerPro\logs`.

## 💡 Feature Requests
Have an idea for a new feature?
Open a **Feature Request** issue on GitHub. Please be detailed about the use case and how it fits into the enterprise-grade vision of the project.

## 💬 Discussions & Q&A
For general questions, troubleshooting, or discussing architecture:
Please use [GitHub Discussions](https://github.com/shovo127/Smart-Scanner-Pro/discussions). Do not open Issues for general questions.
